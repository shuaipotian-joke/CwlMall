# CwlMall


#谷粒商城第一季



### 基础设施

## 🐨 技术栈

| 项目                    | 说明       |
|-----------------------|----------|
| `cwlmall-common`  | 公共依赖     |
| `cwlmall-coupon`     | 优惠模块     |
| `cwlmall-gateway`        | 网关模块     |
| `cwlmall-member`      | 会员模块     |
| `cwlmall-order`       | 订单模块     |
| `cwlmall-product` | 商品模块     |
| `cwlmall-register` | 注册中心     |
| `cwlmall-third-party`  | 第三方模块    |
| `cwlmall-ware`   | 库存模块     |
| `renren-fast`    | 后台系统后端代码 |
| `renren-fast-vue`    | 后台系统前端代码 |
| `renren-generator`    | 人人开源生成器  |
